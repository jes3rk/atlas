Welcome to Atlas

Atlas is a random address generator that pulls from an internal database to generate random, geolocatable addresses within the United States.

Atlas empowers users to populate their web apps with true and accurate test data. Sourcing from Open Addresses, Atlas is an API for querying random addresses. Whether populating test data, testing your geolocation service, or any other use, Atlas has the data.

How To Use Atlas

Atlas is a simple API. The query url is as follows /api/random with a handful of query parameters.
